# VERIDEX Assets

This directory contains visual assets for the VERIDEX project.

## Screenshots

The `screenshots/` folder is intended for dashboard and UI screenshots that demonstrate the system's capabilities:

### Recommended Screenshots

1. **Dashboard Overview** (`dashboard-overview.png`)
   - Main VERIDEX dashboard showing active triage queue
   - Display multiple emails with risk scores and verdicts

2. **Decision Factors Analysis** (`decision-factors.png`)
   - Detailed view of an email showing the Decision Factors breakdown
   - Highlight the transparent XAI reasoning with color-coded factors

3. **Analyst Review Queue** (`analyst-review.png`)
   - Queue of emails requiring analyst review (< 75% confidence)
   - Show the workflow for manual triage

4. **Login Interface** (`login-screen.png`)
   - Clean authentication screen with HIPAA compliance notice

### Screenshot Guidelines

- **Resolution**: Minimum 1920x1080 for clarity
- **Format**: PNG for lossless quality
- **Annotations**: Use arrows/highlights to emphasize key features
- **Redaction**: Remove any sensitive information (emails, usernames, etc.)
- **Consistency**: Use the same theme/data for all screenshots

### Usage in README

Reference screenshots in the main README using:

```markdown
![Dashboard Overview](assets/screenshots/dashboard-overview.png)
```
